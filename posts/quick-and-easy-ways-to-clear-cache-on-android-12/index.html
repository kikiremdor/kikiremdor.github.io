3. Enable cache clearing settings on Android 12#

Android 12 recently released a new feature called System Cache clearing settings. It allows users to clear cache quickly from the home screen, without having to go into the app’s settings. This feature is incredibly useful if you want to free up some storage, or simply reset an app. Here’s a quick guide on how to enable this feature on Android 12.

First, open up the Settings app on your Android device. Scroll down and select Systems. Then, tap on Storage, followed by Cache. This will open the System Cache Clearing settings page.

Once you’re in the System Cache Clearing settings page, select the desired cache clearing option. You can clear all cache from the settings page, or you can select a particular app to clear the cache for. Selecting the app will also open up their respective cache settings page.

Finally, tap on Clear Cache. This will delete all the cached data for all opened apps. Any remaining cached data will be cleared when you restart your device.

That’s all you need to do to enable the System Cache Clearing feature on Android 12! This feature is incredibly useful and can save you a lot of time and effort when you need to clear the cache on your device.
