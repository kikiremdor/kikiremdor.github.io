1. How to take a screenshot on External Keyboard#

Taking screenshots of your screen can be useful for various tasks such as sharing information or illustrating instructions. Microsoft Surface Laptop 3 is equipped with a full-size physical keyboard, which makes it an ideal device for taking screenshots. Here’s a step-by-step guide to taking screenshots on the Microsoft Surface Laptop 3:

First, you need to press the Windows + Print Screen key simultaneously. This will take a screenshot and save it in the Pictures > Screenshots folder.

Second, you need to select the portion of the screen you want to capture. Simply press the Function (Fn) + Space bar button to start “Snip and Sketch” mode. To capture the whole screen, you can simply press Fn + Alt + Space bar.

Third, you can use the arrows to select the desired area of the screen, which will be highlighted in blue. Once you are done, press the Enter key to capture the screenshot.

Finally, you can save the screenshot by clicking the “Save” button at the top of the window. The screenshot will be saved as a .PNG file in the Pictures > Screenshots folder.

I. Taking a Screenshot#

Taking a screenshot is a handy way of sharing info quickly. In this tutorial, we’ll help you learn how to take a screenshot on the Microsoft Surface Laptop 3 using an external keyboard.

Firstly, make sure the external keyboard is connected to your laptop. Then press the FN + Alt + Prt Scr keys to capture the entire screen. To save the screenshot, you can simply press the Windows key + S. This will open the Windows Snipping Tool and save the screenshot to your computer.

Another way to take a screenshot is to press the FN + Windows + Prt Scr keys. You can use this to save the screenshot directly to your OneDrive. You can then easily access the screenshot from any device.

Finally, if you want to capture a specific area, press the Windows + Shift + S keys. This will open a crossing selection box, and you can select the area you want to capture. Once you’ve captured the screenshot, click the Save icon and select the location to save the screenshot.

Taking a screenshot on the Microsoft Surface Laptop 3 is easy and straightforward. With the help of an external keyboard, you can quickly and easily capture any area of your screen and save it to your computer or to the cloud.

II. Sharing the Screenshot#

Taking screenshots on Microsoft Surface Laptop 3 is quite simple, especially when you have an external keyboard. To take a screenshot with an external keyboard, you just need to press the Windows Key + Print Screen Key together. This will take a screenshot of the entire screen and save the image in your pictures folder.

The Windows Key + Shift + S key combination allows you to take a screenshot of a specific area. The screenshot will be saved in the clipboard, and you will need to paste the image in a program such as Microsoft Paint to save it as an image.

If you have the Microsoft Surface Pen, you can use it to take a screenshot by double-clicking the top button on the pen. This will take a screenshot of the current active window, and the image will be saved in the pictures folder.

The final way to take a screenshot on Microsoft Surface Laptop 3 is to use the built-in Windows Snipping Tool. To open the tool, press the Windows Key + R to open the Run box, then type snippingtool and press enter. The snipping tool will open, and you can select from the different options available to take a screenshot. Once you have taken the screenshot, you will be able to save it as an image file.

3. How to take a screenshot with Windows Key Shortcut#

Taking a screenshot on Microsoft Surface Laptop 3 is as easy as pressing two keyboard keys. One of the quickest and most convenient ways to take a screenshot is to use the Windows key shortcut. Here is a step-by-step guide on how to do it.

First, press the Windows key and PrtSc button simultaneously. This will capture the whole screen. The image will be saved in the Pictures folder.

To capture a specific area of the screen, press the Windows key and Shift + PrtSc together. You can then select the area you want to take a screenshot of. The image will be saved in the same folder as before.

If you want to take a screenshot of the active window instead, press the Windows key and Alt + PrtSc at the same time. This will take a screenshot of the window that you currently have open. Again, the image will be saved in the Pictures folder.

As you can see, taking a screenshot on Microsoft Surface Laptop 3 is very easy to do. You just need to remember the Windows key shortcut and you can take screenshots in a matter of seconds.
